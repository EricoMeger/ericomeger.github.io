Bem-vindo!

Olá! Sou Érico Meger, estudante de Ciência da Computação e desenvolvedor de software. Minha experiência abrange desenvolvimento Full-Stack, robótica e ciência de dados, com projetos que vão de aplicações web a sistemas embarcados. Aqui, compartilho alguns dos meus projetos para que você conheça mais sobre meu trabalho e minhas áreas de interesse.

Projetos em destaque:

Portal BCC - Frontend

Projeto desenvolvido em React, com integração a API para exibição dinâmica de conteúdo, como posts, informações do curso e perfil dos professores.

Portal BCC - API

Projeto desenvolvido utilizando Node, TypeScript e MongoDB, seguindo princípios de clean architecture para garantir escalabilidade e manutenção facilitada.

As Raízes do Reino

Jogo sendo desenvolvido em React e Python (FastAPI) para ensinar a raiz quadrada de grafos, gameficando o aprendizado.

LibrePlay - API

API desenvolvida com Node, TypeScript e MySQL, oferecendo endpoints para cadastro, autenticação de usuários e gerenciamento de filmes.

Implementação do algoritmo Pure Pursuit para navegação autônoma de robôs móveis. Desenvolvido utilizando Python e ROS.
